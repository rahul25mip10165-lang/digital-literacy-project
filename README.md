## 📖 Project Overview

This project is part of the Digital Literacy course at VIT Bhopal University. It is designed to develop essential digital skills required for academic and professional growth. The project covers five modules — digital awareness, professional online presence, coding platforms, professional communication, and cybercrime prevention. All work is organized in this GitHub repository, which itself serves as a permanent digital portfolio.
